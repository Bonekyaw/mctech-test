ပါဠိသဒၵါမွာေတာ ့

ပုရိေသာ သယတိ။             ( ကတၱား နည္း + ႀကိယာ နည္း )
ပုရိသာ သယႏၲိ။                    ( ကတၱား မ ်ား + ႀကိယာ မ ်ား )

ျမန္မာျပန္ :    ပုရိေသာ = ေယာက္ ်ားသည္ ၊ သယတိ = အိပ္၏။ ပုရိသာ = ေယာက္ ်ားတို႔သည္ ၊ သယႏၲိ = အိပ္ကုန္၏။

ရွင္းလင္းခ ်က္ :    သယ - သဒၵါသည္ အိပ္ျခင္းအဓိပၸါယ္ ရွိ၏။ အိပ္ရာ ဟူေသာ နာမ္ပုဒ္အသြင္ျဖင့္ ေျပာသည့္အခါ သယနံ ဟုေရးရၿပီး
အိပ္သည္ ဟူေသာ ႀကိယာပုဒ္ အေနျဖင့္ ေျပာသည့္အခါ သယတိ ဟု ေျပာရ ေရးရပါသည္။ တိ ႏွင့္ ဆံုးရင္ အနည္းကိန္း ႀကိယာျဖစ္ၿပီး
ႏၲိ ႏွင့္ ဆံုးပါက အမ်ားကိန္း ႀကိယာ ျဖစ္သြားပါတယ္။ ပုရိေသာ ဟု ၾသ သရျဖင့္ ဆံုးရင္ အနည္းကိန္း နာမ္ပုဒ္ ျဖစ္ၿပီး ပုရိသာ ဟု အာ
သရျဖင့္ ဆံုးပါက အမ်ားကိန္း နာမ္ပုဒ္ျဖစ္သြားပါတယ္။
